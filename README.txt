1. Making HTTP request from python file.
2. Store server side session data in a a shared storage service using the above methodology.-----------------------------------------------------------------------------------------------FILES SUBMITTEDdump.py:  This file demonstrates how to make GET request from python file. Here we calling one of the services of kv(GET) from dump.py.
In this file, we list the current active sessions of KeyValueStore  sessions.py: We need to store the sessions ids in a shared Session Store. For this we are overriding the methods of KeyValueSessionStore to store session ids and count values as key-value pairs in Kv.py.-----------------------------------------------------------------------------------------------INSTALLATION:1.
